Парсер книг с Books to Scrape

Простой парсер на Python для сбора информации о книгах с сайта [Books to Scrape](http://books.toscrape.com/).

Описание:
Этот скрипт собирает данные о книгах с главной страницы сайта Books to Scrape, включая:
- Название книги
- Цену
- Рейтинг
- Ссылку на страницу книги

Собранные данные сохраняются в CSV файл для дальнейшего анализа.

Функционал:
- Парсит список книг с главной страницы
- Извлекает названия и цены
- Определяет рейтинг книг
- Сохраняет данные в CSV формате
- Обрабатывает ошибки запросов
